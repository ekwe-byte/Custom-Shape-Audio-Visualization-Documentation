# 2.4 - Color, Opacity

`Bar Color`: The base RGB color of the visualizer bars
> ⚠️ Must be in RGB format: ($\color{red}{\text{Red}}$, $\color{green}{\text{Green}}$, $\color{blue}{\text{Blue}}$)

Useful websites: 
+ [Google's Color Picker](https://www.google.com/search?q=color+picker)
+ [HTML Color Codes](https://htmlcolorcodes.com/)
+ [Image Color Picker](https://imagecolorpicker.com/)

<br>

`Opacity`: The master transparency level of the bars (0 = invisible, 1 = fully visible).
