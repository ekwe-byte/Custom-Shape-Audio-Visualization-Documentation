## Dynamic Shape

> Format: ```(PREFIX, time_exist, time_morph, ease, loop) => ... => (...)```

+ Loop: 0 - Open shape
+ Loop: 1 - Close shape

Example: ```(EX_LINE_, 1, 0.5, 6, 0) => (EX_TRGL_, 1, 0.5, 6, 1) => (EX_HRT_, 1, 0.5, 6, 1)```
